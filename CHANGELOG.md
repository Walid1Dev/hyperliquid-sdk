# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-14

### Added
- Initial release
- `HyperliquidClient` class with WebSocket connection
- Real-time price subscriptions (all assets or single)
- Order book subscriptions with depth
- Trade stream subscriptions
- Candle/chart data subscriptions
- User data (positions, orders, fills) with wallet authentication
- Auto-reconnection with exponential backoff
- Full TypeScript type definitions
- ESM and CommonJS dual exports
- Comprehensive documentation
